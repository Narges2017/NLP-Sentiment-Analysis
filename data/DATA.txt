# Data

This project uses the **IMDB Dataset of 50K Movie Reviews** from Kaggle.

## Dataset details

- **Source**: [IMDB Dataset of 50K Movie Reviews on Kaggle](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews)
- **Size**: 50,000 movie reviews
- **Columns**: 
  - `review`: Text of the movie review
  - `sentiment`: Label (positive or negative)
- **Class distribution**: Balanced (25,000 positive, 25,000 negative reviews)

## How to use

To reproduce the analysis:

1. Download the dataset from Kaggle (requires free Kaggle account)
2. Extract `IMDB Dataset.csv` 
3. Place it in this `data/` folder, or attach it as input when running the notebook on Kaggle

**Note**: The CSV file is not included in this repository due to size constraints. Data files are excluded via `.gitignore`.
